Introducción.

Despues de instalar Oracle 11g R2, con el asistente de instalacion, y creada la BD para SOA suite.

Instalación de schema para SOA.

Se debe descargar SOA WebLogic, SOA Suite y RCU para SAO (e instalar en este orden). Una vez instalado en este orden, se creara el esquema para SOA en Oracle 11g :

$HOME/rcuHome/bin/rcu

Se proporcionan los datos de conexion y se usa el usuario SYS con rol SYSDBA. Una vez validada la conexion, se debera seleccionar SOA AND BPM infraestructure. Se proporciona un password para todos los esquemas.

Creacion de dominio SOA en weblogic.

Para crear el dominio SOA en weblogic (se debe previamente haber instalado SOA suite) se debe ejecutar :

$HOME/Oracle/Middleware/wlserver_10.3/common/bin/config.sh
